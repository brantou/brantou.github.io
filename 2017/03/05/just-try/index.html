1.1 org-mode的文档结构

下面这段摘录自org-mode手册, TAB 用于大纲的展开和折叠，让你轻松的定位于工作，撇除其他无关干扰。

Outlines allow a document to be organized in a hierarchical structure, which (at least for me) is the best representation of notes and thoughts. An overview of this structure is achieved by folding (hiding) large parts of the document to show only the general document structure and the parts currently being worked on. Org greatly simplifies the use of outlines by compressing the entire show/hide functionality into a single command, org-cycle, which is bound to the TAB key.

标题定义了大纲树的结构,以处于一行左边缘的一个或多个星号开头,如下: 

* Top level headline
** Second level
*** 3rd level
     some text
*** 3rd level
     more text
* Another top level headline
